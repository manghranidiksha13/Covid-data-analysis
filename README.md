# COVID-19 Patient Symptoms Analysis

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (SQLite / PostgreSQL compatible)

## Project Overview
This project analyzes COVID-19 patient symptom data to identify
high-risk groups, symptom trends, and demographic patterns.

## Steps Performed
- Data cleaning and preprocessing using Python
- Exploratory Data Analysis (EDA)
- SQL-based business queries using SQLite
- Actionable healthcare insights

## Key Insights
- Fever and fatigue are the most common symptoms
- Senior patients show higher risk
- Patients with multiple symptoms need priority attention
